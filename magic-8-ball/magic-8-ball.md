### Introduction

Re-create this fortune telling toy in ruby!


### Learning Goals

* Work with the Array data structure
* Read Ruby Documentation
* Loading code into irb
* Calling methods on Arrays
  - square-brackets `[]`
  - `.first`
  - `.last`
  - `.size` and `.length`
  - `.shuffle`
  - `.sample`
  - the shovel operator `<<`
